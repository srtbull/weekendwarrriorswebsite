Advanced MLRS

Plugin Documentation for Rust (Oxide/uMod)

Version 4.5.0 · Author: Banshee

Created by Banshee for enjoyment on The Weekend Warrior Server Group
Overview

Advanced MLRS lets players call in an MLRS (Multiple Launch Rocket System) strike by throwing a special designator grenade. The grenade does not explode; it acts as a marker. When the fuse would have expired, rockets rain down on that location with the authentic MLRS whistle sound. Permission levels control how many rockets each player gets, and admin permission grants double damage.

Requirements
  • Rust server with Oxide or uMod installed.
  • Permission to use custom skins on your server (skin system enabled).
  • A reskinned F1 grenade used as the designator (see Skin Setup below).
Skin Setup (Required)

The plugin uses a reskinned F1 grenade as the designator (not binoculars or a separate item). The skin must match the plugin’s expected skin ID.

Workshop Skin ID: 841650807

Your server must allow this skin on the F1 grenade. Without it, the designator will not be recognised and the strike will not trigger. To use a different skin, edit the plugin source and change DesignatorSkinId to the desired workshop ID, then recompile.

For Server Owners — Loot Tables & Kits

Add this item to your loot tables or kit plugins so players can get the designator:

Item shortname: grenade.f1
Amount: 1
Skin ID: 841650807

Use your loot or kit plugin’s syntax to add grenade.f1 with skin 841650807. Normal F1 grenades are unchanged; only this skinned version is the designator.

How Players Get the Designator

The plugin does not give the designator. The server owner adds it via:

  • Loot table — Add the skinned F1 to crates, NPCs, or other loot.
  • Kits — Include the same skinned F1 in a VIP or reward kit.

Players then throw the designator like a normal F1 to call the strike.

How to Use (Players)
  1. Obtain the designator from loot or a kit.
  2. Throw the designator at the target location (like a normal F1).
  3. The grenade does not explode. After about 2.5 seconds it disappears and triggers the MLRS strike. Rockets fall from above with the MLRS whistle and explode on impact.
  4. How many rockets fire depends on the player’s permission (see below).

Dependency on Reskinned Grenade
  • Item: F1 Grenade (grenade.f1)
  • Skin ID: 841650807
  • Behaviour: When a player throws an F1 with this skin, the plugin disables its explosion and uses it only as a marker.

Ensure your server allows workshop skin 841650807 for the F1 grenade.

Quick Reference

Troubleshooting

No designator / wrong item
The designator is an F1 with skin ID 841650807. The server owner must add it to loot or kits. If the skin is blocked, enable it for grenade.f1.

Grenade explodes instead of calling a strike
The thrown grenade must have the exact designator skin (841650807). Normal F1s still explode.

No rockets / no MLRS sound
Ensure your game/server build includes MLRS content (e.g. ammo.rocket.mlrs).

Admin double damage not applying
Confirm the thrower has advancedmlrs.admin and no other plugin is overriding damage.
